)
static

Logs the string $logString to the logfile webservices.log in the current log directory (usually var/log).

If logging is disabled, nothing is done.

In dev mode, also writes to the eZP logs to ease debugging (this happens regardless of the logging level set for the extension itself)

This is implemented here because . it relies on the templating system . it does its own caching but it should really be a function of the server class itself. Since we do not want to pollute server classes with eZ specifics, we chose to avoid the reverse depenency by putting the code here.
