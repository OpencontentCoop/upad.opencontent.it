Detailed Description

Base class for handling all incoming webservice calls.

API based on the eZSOAPServer class from eZP. See docs in there.

Process of handling incoming requests:

processRequest | – parseRequest (builds a request obj out of received data) | – handleRequest or handleInternalRequest (builds response as plain php value) | – showResponse (echoes response in correct format, usually via building a response obj and serializing it) | – prepareResponse (adds to response obj any missing info gathered usually during parseRequest)
