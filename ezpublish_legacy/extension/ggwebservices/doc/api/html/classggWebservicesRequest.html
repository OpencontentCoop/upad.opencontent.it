ggWebservicesRequest::requestURI (   $uri)

Returns the final URI that will be used by the client, based on its initial version (except the protocol://host:port part).

Useful for protocols that encode parameters in the URL and use GET instead of POST. Default is not to touch the url received and give it back intact. Override it in case of need.
